# Social Links Profile

## Descrizione
Progetto di una **card profilo con link social**, sviluppato in HTML5 e CSS3 come esercizio di stile e responsive design.  
La card mostra avatar, nome, località, breve bio e una lista di link ai social principali.  

---

## Tecnologie
- HTML5 con struttura semantica  
- CSS3 (Flexbox, inline-block, min/max width)  
- Google Fonts (Inter)  
- Palette colori in HSL per consistenza e accessibilità  

---

## Funzionalità
- Card centrata verticalmente e orizzontalmente con Flexbox  
- Avatar circolare e tipografia coerente  
- Bottoni social con hover effect e layout responsive  
- Larghezza minima (`min-width: 320px`) per evitare accavallamenti su schermi piccoli  

---

## Istruzioni
1. Clona la repository o scarica i file.  
2. Apri `index.html` in un browser moderno.  
3. Personalizza `style.css` per colori, spaziature o font.
